# CSV数据展示页面

## 功能概述

CSV数据展示页面提供了项目1和项目2电池数据的可视化展示功能，支持：

- 项目数据概览和统计
- 数据质量指标展示
- Bank详细信息表格
- 实时数据加载和刷新

## 主要组件

### CsvDataPage
主要的页面组件，包含：
- 数据加载和状态管理
- 项目卡片展示
- Bank数据表格
- 数据质量可视化

### CsvDataService
数据服务层，负责：
- 连接CSV转换器和Web界面
- 数据格式转换
- 错误处理和日志记录

## 数据流程

1. **数据扫描**: 扫描项目1和项目2的文件结构
2. **数据解析**: 解析CSV文件内容
3. **数据转换**: 转换为标准化格式
4. **数据验证**: 进行质量检查和异常检测
5. **数据展示**: 在Web界面中可视化展示

## 使用方法

1. 在侧边栏点击"CSV数据展示"
2. 系统自动加载项目1和项目2数据
3. 查看数据概览和质量指标
4. 点击项目卡片查看详细信息
5. 使用刷新按钮更新数据

## 配置说明

当前使用硬编码的数据路径：
- 项目1: `../../../项目1`
- 项目2: `../../../项目2`

实际部署时需要配置正确的数据路径。

## 注意事项

- 数据加载可能需要一些时间，请耐心等待
- 如果数据路径不存在，会显示相应的警告信息
- 数据质量指标基于实际数据计算，可能因数据质量而变化
