## 1. 项目概述

### 1.1 项目背景
全球储能平台POC是一个面向储能项目管理的综合性Web平台，旨在为储能项目提供统一的数据管理、监控分析和运维支持。平台支持多项目、多层级的电池系统管理，从全球总览到单体电芯的精细化管理。

### 1.2 项目目标
- 提供统一的储能项目数据管理平台
- 实现多层级的电池系统监控（项目→Bank→电芯）
- 支持多种数据源的接入和处理
- 提供智能化的电池健康分析和告警管理
- 为运维决策提供数据支持

## 2. 当前功能模块

### 2.1 全球总览 (L1级)
**功能描述：** 提供全球储能项目的宏观视图和关键指标监控

**核心功能：**
- **项目统计概览**
  - 总项目数量统计
  - 总Bank数量统计
  - 总容量和总功率展示
  - 实时数据更新

- **项目列表管理**
  - 项目基本信息展示
  - 项目状态监控（正常/警告/异常）
  - 项目关键参数（电压、电流、SOC、SOH）
  - 快速项目导航

- **系统状态监控**
  - 正常/警告/异常项目数量统计
  - 今日充放电量统计
  - 系统运行状态总览

### 2.2 项目管理 (L2级)
**功能描述：** 针对具体储能项目的详细管理和监控

**核心功能：**
- **项目详情管理**
  - 项目基本信息维护
  - 项目配置参数管理
  - 项目运行历史记录

- **项目列表视图**
  - 支持项目筛选和搜索
  - 项目状态批量管理
  - 项目性能对比分析

### 2.3 样本数据展示 (L3级)
**功能描述：** 展示和分析来自不同数据源的电池样本数据

**核心功能：**
- **项目1数据管理**
  - 支持2#、14#、15#系统数据
  - 中文表头识别和解析
  - 单体电池数据处理（V1-V400, T1-T400, SOC1-SOC400, SOH1-SOH400）

- **项目2数据管理**
  - 支持group1-4组数据
  - 多数据类型处理（voltage, soc, temperature, state）
  - 时间序列数据分析

- **数据质量评估**
  - 完整性评估（数据缺失率）
  - 准确性评估（数据合理性）
  - 一致性评估（数据一致性）
  - 及时性评估（数据时效性）

- **数据可视化**
  - 项目数据概览卡片
  - 数据质量进度条展示
  - Bank详细信息表格
  - 实时数据刷新

### 2.4 电池分析中心 (L4级)
**功能描述：** 专业的电池健康状态分析和智能诊断

**核心功能：**
- **电池一致性分析**
  - 电压一致性评分（基于最大差异和标准差）
  - 温度一致性评分（温度分布均匀性）
  - SOC一致性评分（荷电状态一致性）
  - 综合一致性评分计算

- **电池容量分析**
  - 额定容量vs实际容量对比
  - 容量保持率计算（85%-95%范围）
  - 能量容量评估（kWh）
  - 可用容量分析（90%可用率）
  - 容量衰减率预测（2%-5%/年）

- **智能告警管理**
  - 多级别告警（信息/警告/错误/严重）
  - 多类型告警（电压/温度/SOC/SOH/一致性/容量）
  - 告警统计和趋势分析
  - 告警处理状态跟踪

- **统计分析**
  - 平均一致性评分计算
  - 平均容量保持率统计
  - 告警数量统计
  - 严重告警监控

### 2.5 电池组管理 (L5级)
**功能描述：** Bank级别的电池组管理和监控

**核心功能：**
- Bank状态监控
- Bank性能参数管理
- Bank级别的告警处理

### 2.6 电芯管理 (L6级)
**功能描述：** 单体电芯级别的精细化管理

**核心功能：**
- 单体电芯参数监控
- 电芯性能分析
- 电芯故障诊断

## 3. 技术架构

### 3.1 前端技术栈
- **框架：** React 18 + TypeScript
- **UI组件：** Ant Design
- **路由：** React Router
- **状态管理：** React Hooks
- **样式：** Tailwind CSS
- **构建工具：** Vite

### 3.2 数据处理引擎
- **CSV解析：** Papa Parse
- **文件监控：** Chokidar
- **数据验证：** 自定义验证器
- **错误处理：** 分层错误处理机制
- **日志系统：** Winston日志框架

### 3.3 核心特性
- **响应式设计：** 支持多设备访问
- **实时数据：** 动态数据刷新
- **模块化架构：** 可扩展的组件设计
- **国际化支持：** 中英文界面
- **错误恢复：** 完善的错误处理机制

## 4. 数据处理能力

### 4.1 支持的数据格式
- **CSV文件：** 标准CSV格式
- **中文表头：** 支持中文字段识别
- **多种编码：** UTF-8, GBK等
- **大文件处理：** 支持大容量数据文件

### 4.2 数据处理流程
1. **文件扫描：** 自动扫描指定目录
2. **格式识别：** 自动识别文件格式和项目类型
3. **数据解析：** 解析CSV数据并验证
4. **数据转换：** 转换为标准格式
5. **质量检查：** 数据质量评估
6. **存储展示：** 数据存储和可视化展示

### 4.3 数据质量保证
- **完整性检查：** 检测数据缺失
- **合理性验证：** 验证数据范围
- **一致性校验：** 检查数据一致性
- **异常检测：** 识别异常数据点
